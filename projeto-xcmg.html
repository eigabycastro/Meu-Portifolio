Projeto: Sistema de Gestão Integrada XCMG

Desenvolvido por Gabrielly Castro - Desenvolvedora Front-End

Descrição do Projeto

O Sistema XCMG é uma aplicação web integrada para gerenciamento de processos empresariais na XCMG Brasil. Ele abrange módulos para controle financeiro, registro de horas para a Lei do Bem, gestão de traduções de manuais técnicos e visualização de dados. O projeto foi construído com foco em eficiência, usabilidade e integração com serviços externos como Google Sheets e envio de emails via Gmail.

Como desenvolvedora front-end, contribui com a criação de interfaces intuitivas e responsivas, utilizando HTML5, CSS3 e JavaScript vanilla. O backend foi implementado em Python com Flask, incluindo caching, logging e integração com APIs do Google.

Este sistema otimiza fluxos de trabalho, como registro de pagamentos, cálculo de horas e acompanhamento de traduções, promovendo maior produtividade e conformidade fiscal.

Tecnologias Utilizadas

Python (Flask) HTML5 CSS3 (com variáveis e gradientes) JavaScript (DOM manipulation, events) Google Sheets API (gspread) Flask-Mail Material Icons Responsive Design (Media Queries) Caching e Threading

Funcionalidades Principais

  • Módulo Financeiro: Registro de pagamentos, despesas, anexos de arquivos (PDF, imagens) e integração com Lei do Bem.
  • Módulo Lei do Bem: Registro de horas em projetos de P&D, cálculo automático de horas e validação de campos.
  • Módulo de Traduções: Gerenciamento de traduções de manuais, incluindo idiomas, páginas, laudas e valores.
  • Visualização de Registros: Filtros avançados, tabela dinâmica, exportação de dados e integração com dashboard Power BI.
  • Funcionalidades Gerais: Envio de emails automáticos, caching para performance, validação de formulários, upload de arquivos e design glassmorphism.
  • APIs de Debug: Endpoints para teste, verificação de colunas e registros recentes.
